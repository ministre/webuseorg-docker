1. Установка с нуля

- скопировать все файлы в папку сайта
- запустить http://имя_сайта/install.cfg
- поправьте config.php


2. Импорт из программы useorg

- скопировать все файлы в папку сайта
- создать БД
- залить в неё базу webuseorg.sql
- поправьте config.php
- запустите /install/import.php
- введите имя базу откуда будем брать данные, нажмите "Начать"
- в phpmyadmin поправьте mode=1 для пользователя который будет админом. Возможно нужно будет поправить принадлежность пользователя к организации.
